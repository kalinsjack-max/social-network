# Social Network (Facebook Clone)

A full-stack social media platform with real-time messaging, news feed, and friend management.

## Features

- ✅ User Authentication (JWT)
- ✅ News Feed with Posts, Likes, Comments, Shares
- ✅ Real-time Messaging (Socket.io)
- ✅ Friend Requests System
- ✅ Notifications
- ✅ Profile Management
- ✅ Responsive Design

## Tech Stack

**Backend:**
- Node.js
- Express.js
- MongoDB (Mongoose)
- Socket.io
- JWT Authentication
- bcrypt.js

**Frontend:**
- HTML5
- CSS3 (Custom)
- Vanilla JavaScript
- Socket.io Client
